# Changelog - 08.02.2026

## 🎉 Основные изменения

### 🏭 Производитель схем
- **Добавлено поле "Manufacturer"** в модель Project
- **Справочник производителей**: Dimensions, DMC, Anchor, Bucilla, Janlynn, Mill Hill, RIOLIS, Heritage Crafts, Vervaco, Lanarte, Panna, Русская сказка, Алиса, Чудесная игла
- **Свой вариант**: Возможность ввести производителя вручную (кнопка "Other")
- **Отображение**: Производитель показывается на странице проекта

### 📊 Статистика "Days"
- **Заменено "Sessions" на "Days"** в статистике
- **Подсчет дней**: Количество дней с момента первого лога
- **Дополнительно**: Внизу указано количество логов

### 📝 Упрощенная форма лога
- **Ручной ввод**: Простое поле для ввода количества крестиков за день
- **Автоматический расчет**: Total = Previous + Daily (автоматически)
- **Опциональное фото**: Загрузка фото прогресса работы
- **Заметки**: Поле для текстовых заметок
- **Убран AI подсчет**: Пока не используется (для будущих версий)

### 📸 Обработка изображений
- **Автоматическая оптимизация** при загрузке:
  - Resize до макс. 1920px (качество 85%)
  - Создание thumbnail 800px (качество 80%)
  - Конвертация в WebP формат
  - Уменьшение размера в 10-20 раз
- **Увеличен лимит**: до 10MB (до обработки)

### 🔍 Модальное окно для фото
- **Маленькие превью**: 80x80px в логах, 128x128px на странице проекта
- **Клик для увеличения**: Открывается полноразмерное фото в модальном окне
- **Hover эффект**: "Click to enlarge" при наведении
- **Закрытие**: Кнопка X или клик вне изображения
- **Работает для**: Фото схемы и фото прогресса

### 🗑️ Быстрое удаление
- **Иконка корзины** в правом верхнем углу карточек проектов в Dashboard
- **Диалог подтверждения** перед удалением
- **Без захода в проект**: Удаление прямо из списка

### 🛠️ Технические улучшения
- **Обратная совместимость**: Поддержка старых и новых форматов данных
- **Защита от ошибок**: Проверки на null для всех полей
- **Исправлен график**: Правильная обработка старых логов (stitches → totalStitches, dailyStitches)
- **Исправлено отображение фото**: schemaImage и photoUrl

### 🗄️ База данных
- **Миграция**: `add_manufacturer` - добавлено поле manufacturer в Project

## 🔄 Миграция данных

Проект поддерживает обратную совместимость:
- Старые проекты без `manufacturer` - работают нормально
- Старые логи с `stitches` - автоматически конвертируются в `totalStitches`/`dailyStitches`
- Старые фото с `imageUrl` - работают как `schemaImage`/`photoUrl`

## 🚀 Как использовать новые функции

### Производитель
1. Создайте или отредактируйте проект
2. Выберите производителя из списка
3. Или нажмите "Other" и введите свой вариант

### Добавление лога
1. Откройте проект → "Add Daily Log"
2. Введите количество крестиков, вышитых за день
3. (Опционально) Загрузите фото работы
4. (Опционально) Добавьте заметки
5. Нажмите "Save Log"

### Просмотр фото
1. Наведите курсор на любое фото
2. Увидите "Click to enlarge"
3. Кликните → откроется полноразмерное изображение
4. Закройте кнопкой X или кликом вне фото

### Быстрое удаление
1. Откройте Dashboard
2. Наведите на карточку проекта
3. Нажмите иконку корзины в правом верхнем углу
4. Подтвердите удаление
