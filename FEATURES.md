# StitchArena - Функции

Полный список реализованных функций приложения для отслеживания вышивки крестиком.

## 📊 Tracking & Analytics

### Ежедневное отслеживание
- ✍️ Ручной ввод прогресса
- 📸 Загрузка фото работ
- 📝 Заметки к логам
- 📊 Автоматический подсчет крестиков

### Статистика
- 📈 **Графики прогресса** (Recharts)
  - Общий прогресс проекта
  - Ежедневные данные
  - Средняя скорость
- 📊 **Общая статистика на Dashboard**
  - Всего проектов (активных, завершенных, на паузе)
  - Общее количество крестиков
  - Средняя скорость за 6 месяцев
  - Самый продуктивный день/неделя
  - Top 5 проектов
- 📅 **Activity Heatmap**
  - GitHub-style календарь
  - 12 месяцев истории
  - Цветовая кодировка по количеству
- ⏱️ **Прогноз завершения**
  - Ожидаемая дата завершения
  - На основе средней скорости
  - Отображается в карточках проектов

### Стрики
- 🔥 **Текущий стрик** - дни подряд с логами
- 🏆 **Рекордный стрик** - максимальный стрик
- ✨ Grace period - засчитывается если есть лог сегодня ИЛИ вчера

## 🎮 Gamification

### Достижения (17 штук)
Разбиты по 6 категориям:

#### 🌱 First Steps (Первые шаги)
- 🎯 **First Project** - создать первый проект
- 💯 **First 100** - вышить первые 100 крестиков
- 🎊 **First 1000** - вышить первые 1,000 крестиков
- 🎉 **First Complete** - завершить первый проект

#### ⚡ Consistency (Постоянство)
- 📅 **Week Warrior** - стрик 7 дней
- 🗓️ **Month Master** - стрик 30 дней
- 🔥 **Streak Legend** - стрик 100 дней

#### 🚀 Speed Demon (Скорость)
- ⚡ **Speed Demon** - средняя скорость 200+ крестиков/день
- 🚀 **Lightning Fast** - средняя скорость 500+ крестиков/день

#### 🎯 Completionist (Завершение)
- ✅ **Finisher** - завершить 5 проектов
- 🎭 **Project Juggler** - 3+ активных проекта одновременно
- 💎 **Perfectionist** - завершить проект 20,000+ крестиков

#### 💬 Social Butterfly (Социальное)
- ❤️ **First Like** - получить первый лайк
- 💬 **Commenter** - написать 10 комментариев
- 🌟 **Popular Project** - 50+ лайков на проекте

#### 🏅 Milestones (Вехи)
- 🏔️ **Big Project** - проект 50,000+ крестиков
- 🗻 **Mega Project** - проект 100,000+ крестиков

### Speed Tiers (Уровни скорости)
Рассчитываются на основе средней скорости за последние 6 месяцев:

- 🐢 **Turtle** (Черепашка) - 0-50 крестиков/день
- 🚴 **Bike** (Велосипед) - 51-150 крестиков/день
- 🚗 **Car** (Машина) - 151-300 крестиков/день
- 🚀 **Rocket** (Ракета) - 301+ крестиков/день

### UI
- 🎨 Компактные значки с градиентами
- 📊 Progress bar для отслеживания прогресса
- 🏷️ Категории с табами
- ✨ Gold градиент для разблокированных, gray для заблокированных

## 👥 Social Features

### Likes (Лайки)
- ❤️ **Кнопка лайка** на проектах
- 📊 **Счетчик лайков**
- ⚡ **Toggle** - одним кликом лайк/анлайк
- 🔄 **Оптимистичные обновления** UI
- ⭐ **Страница Favorites** - все лайкнутые проекты

### Comments (Комментарии)
- 💬 **Комментирование** публичных проектов
- ✍️ **До 1000 символов**
- 🗑️ **Удаление** своих комментариев
- 👤 **Аватары** пользователей
- ⏰ **Временные метки** (date-fns)
- 📍 **Только на публичных** проектах

### Community Feed (Лента активности)
- 🌟 **Объединенная лента** новых проектов и обновлений
- 🔍 **3 фильтра**:
  - All Activity - всё
  - New Projects - только новые проекты
  - Progress Updates - только обновления прогресса
- 🔄 **Infinite scroll** с пагинацией
- 📊 **Два типа карточек**:
  - **ProjectFeedCard** - новые проекты с изображением, прогрессом, темами
  - **LogFeedCard** - обновления с фото и количеством крестиков
- ⚡ **Load More** кнопка

### Public Gallery (Галерея)
- 🖼️ **Публичные проекты** от всех пользователей
- 🏷️ **Фильтрация по темам**
- 📊 **Сортировка** (newest, progress)
- 🎨 **Hover эффекты**
- ❤️ **Лайки** прямо из галереи
