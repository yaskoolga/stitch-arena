## 📋 Следующие социальные функции

### 🔥 Высокий приоритет

#### 1. Лента активности подписок
**Описание:** Показывать обновления от пользователей и проектов, на которые подписан пользователь

**Задачи:**
- [ ] Создать страницу `/feed` с фильтрами
- [ ] Фильтр "Все" / "Только подписки" / "Только проекты"
- [ ] Показывать новые логи от подписанных проектов
- [ ] Показывать новые проекты от подписанных пользователей
- [ ] Маркер "NEW" для непросмотренных обновлений

**Польза:** Пользователи будут активнее возвращаться, чтобы посмотреть обновления

---

#### 2. Коллекции / Сохранённое
**Описание:** Возможность сохранять проекты в папки-коллекции

**Задачи:**
- [ ] Модель `Collection` (название, описание, userId, isPublic)
- [ ] Модель `CollectionItem` (связь с проектами)
- [ ] Кнопка "Сохранить" на карточках проектов
- [ ] Страница `/collections` со списком коллекций
- [ ] Страница `/collections/[id]` с проектами в коллекции
- [ ] Предустановленные коллекции: "Избранное", "Вдохновение", "Хочу повторить"

**Польза:** Организация понравившихся проектов, повторное использование

---

#### 3. Репосты / Шеринг проектов
**Описание:** Поделиться проектом в своей ленте или через ссылку

**Задачи:**
- [ ] Модель `Share` (userId, projectId, comment, createdAt)
- [ ] Кнопка "Поделиться" на проектах
- [ ] Показывать репосты в ленте
- [ ] Копирование ссылки с preview (Open Graph)
- [ ] Счётчик репостов на проектах

**Польза:** Виральность, распространение проектов

---

### 🌟 Средний приоритет

#### 4. Расширенные реакции
**Описание:** Эмоджи-реакции вместо просто лайков (как в Slack/Discord)

**Задачи:**
- [ ] Модель `Reaction` (userId, projectId, emoji)
- [ ] UI для выбора реакции
- [ ] Показывать все реакции под проектом
- [ ] Анимация при добавлении реакции

**Эмоджи:** 😍 🔥 👏 🎉 ✨ 💯 ❤️ 🤩

**Польза:** Больше способов выразить эмоции, геймификация

---

#### 5. Упоминания в комментариях
**Описание:** Возможность упоминать @username в комментариях

**Задачи:**
- [ ] Парсинг @mentions в тексте комментариев
- [ ] Автокомплит при вводе @
- [ ] Уведомления при упоминании
- [ ] Подсветка упоминаний в тексте

**Польза:** Прямые обращения, вовлечение в обсуждения

---

#### 6. Социальная статистика в профилях
**Описание:** Расширенная информация о пользователе

**Задачи:**
- [ ] Показывать количество подписчиков/подписок
- [ ] Вкладки: "Проекты" / "Подписчики" / "Подписки"
- [ ] Топ проектов пользователя (по лайкам)
- [ ] График активности (commit-style heatmap)

**Польза:** Понимание популярности, мотивация
